# Tailwind CSS Demo Page

This project is a simple HTML page demonstrating the usage of **Tailwind CSS** utility classes for styling various elements such as padding, margin, width, height, font properties, and responsive design.

---

## Features

### Padding
- Demonstrates padding using Tailwind's `p-*` classes.
- Example: `<div class="p-10 bg-green-400 text-center">p-20</div>`

### Margin
- Demonstrates margin using Tailwind's `m-*` classes.
- Example: `<div class="m-20 bg-yellow-400">m-5</div>`

### Width
- Fixed width using `w-*` classes.
- Min-width and max-width examples using `min-w-*` and `max-w-*`.
- Example: `<div class="w-96 h-24 bg-black text-white p-6 rounded">this is width element</div>`

### Height
- Fixed height using `h-*` classes.
- Min-height and max-height using `min-h-*` and `max-h-*`.
- Example: `<div class="h-44 w-44 bg-black p-12 text-white text-center">height of the element</div>`

### Font Family
- Using Tailwind fonts like `font-serif` and `font-mono`.
- Example: `<p class="p-[20px] bg-yellow-300 mt-12 font-mono">Lorem ipsum...</p>`

### Font Size
- Tailwind font size classes: `text-xs`, `text-sm`, `text-base`, `text-lg`, `text-xl`, `text-2xl`, etc.
- Responsive font sizes using `sm:`, `md:`, `lg:` prefixes.
- Example: `<p class="text-base sm:text-lg md:text-xl lg:text-2xl">Responsive text size</p>`

### Font Style
- Italic style using `italic`.
- Example: `<p class="mt-12 ml-12 italic">Lorem ipsum...</p>`

### Font Weight
- Demonstrates all font weights from `font-thin` to `font-black`.
- Responsive font weight using `md:font-bold`, `lg:font-extrabold`.
- Example: `<p class="font-normal md:font-bold lg:font-extrabold">Responsive font weight</p>`
